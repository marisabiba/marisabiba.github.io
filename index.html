Trip Planner iOS App

Project Overview

Trip Planner is a modern iOS application built with SwiftUI that helps users plan and organize their trips effectively. This is a personal project, fully designed and developed by me, covering all aspects from UI/UX to architecture, persistence, and feature implementation.

Features

  • Create, edit, and delete trips with start/end dates
  • Organized trip list view
  • Day-by-day planning with chronological activities
  • Add activity details: time, location, notes
  • Track transportation arrangements per day
  • Budget tracking for each day and planned expenses
  • Checklists for daily tasks and packing

Technical Details

  • Framework: SwiftUI
  • Architecture: MVVM (Model-View-ViewModel)
  • Data Persistence: UserDefaults
  • Minimum iOS Version: iOS 16.4
  • Swift Version: 5.0

Personal Contributions

  • Designed and implemented all UI/UX using SwiftUI
  • Developed MVVM architecture for maintainability
  • Implemented custom data models for trips, days, activities, budgets, and checklists
  • Handled all data persistence and state management
  • Created custom views for activity scheduling, budget input, and checklist tracking
  • Tested app on multiple devices and iOS versions
  • Iterated on design based on personal use and feedback

Growth & Learning

  • Deepened knowledge of SwiftUI and MVVM
  • Improved skills in iOS app architecture and state management
  • Enhanced UI/UX design abilities for mobile
  • Learned to optimize for performance and user experience
