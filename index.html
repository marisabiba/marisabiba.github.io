Project Overview

Technologies Used

Spring Boot Spring Security JWT H2 Database JUnit 5 Mockito Swagger

Key Features

  • Secure authentication with JWT tokens
  • Account creation and management
  • Money transfer between accounts
  • Transaction history and statements
  • Balance inquiry and account details
  • Fraud detection algorithms
  • API rate limiting and security headers
  • Comprehensive unit and integration tests

My Role & Responsibilities

  • Solo backend developer for the entire API development
  • Researched and implemented banking security standards (PCI DSS)
  • Designed database schema for account and transaction management
  • Developed JWT-based authentication and authorization system
  • Implemented fraud detection algorithms using machine learning
  • Created comprehensive API documentation using Swagger
  • Built automated testing suite with 95% code coverage
  • Implemented rate limiting and security headers
  • Conducted security audits and penetration testing
  • Optimized API performance handling 1000+ concurrent requests

API Endpoints

  • POST /api/auth/login - User authentication
  • GET /api/accounts - Get account information
  • POST /api/transactions/transfer - Transfer funds
  • GET /api/transactions/history - Transaction history
  • GET /api/accounts/balance - Check balance
